# Sigmacraft Datapacks
This is repository of datapacks used on Sigmacraft Season 10 (Version 1.21.10)

## AFK Farms
A Datapack to spawn fake players from Carpet Mod to AFK at our farms

## Better Loot Tables
A collection of custom loot tables

## Better Recipes
A collection of custom recipes

## Make Blast Furnaces Great Again
A datapack to make Blast Furnaces so much better, originally by [Mathmagician8191](https://github.com/Mathmagician8191), but tweaked for Sigmacraft Season 10.
